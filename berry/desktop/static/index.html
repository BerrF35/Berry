💻 Local Model (Free & Private)

Run models directly on your hardware using Ollama. No data leaves your machine.

Local Setup

  1. Download and install Ollama.
  2. Open your terminal and run the following command to download the recommended model:
ollama run qwen2.5-coder:7b

Note: If you are an advanced user, you can use any local model of your choice by specifying its Ollama tag later in the Settings menu.

Once it finishes downloading and says 'success', you can click complete.
